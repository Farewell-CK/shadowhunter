# 猎影 (ShadowHunter) - 视频语义检索系统

基于智谱 AI 的全栈视频语义检索系统，专为视频侦查设计。

## 核心特性 (针对公安实战优化)

- **智能事件扫描**: 采用先扫描再切片的策略。利用 YOLO 模型快速定位目标出现区间并进行智能聚类，彻底解决固定切片导致的冗余问题。
- **三级级联过滤**: 
    1. **OpenCV 运动检测**: 毫秒级剔除 100% 静止的无效画面。
    2. **Local YOLO 初筛**: 精准拦截无目标（人/车）的画面，算力资源利用率提升 10 倍。
    3. **GLM-4.6V 深度理解**: 对高价值片段提取图侦级结构化特征。
- **语义提纯向量化**: 强制 AI 输出高浓度实词组合，极大提升复杂案情描述下的检索确凿性。
- **超大视频流式处理**: 针对 GB 级长监控视频，支持流式分块分析与断点续传。

## 功能特性

- **智能扫描**: 先扫描后切片，精准定位目标时段，解决冗余分析
- **视觉理解**: GLM-4.6V 视频分析，精准识别人物、车辆、行为
- **语义检索**: 自然语言搜索，秒级定位目标视频片段
- **视觉核实**: Top-N 结果二次核验，生成专业分析报告

## 快速开始

### 1. 配置 API Key

编辑 `backend/config.py`，设置智谱 API Key：

```python
ZHIPU_API_KEY = "your_api_key_here"
```

### 2. 启动服务

**Linux/macOS:**
```bash
chmod +x start.sh
./start.sh --install
```

**Windows:**
```cmd
start_windows.bat
```

### 3. 访问系统

- 前端界面: http://localhost:3000
- API 文档: http://localhost:8000/docs

## 使用流程

1. **配置过滤参数**: 在前端开启“智能扫描”或“运动检测”，设置感兴趣目标（如：person）。
2. **上传视频**: 支持长视频上传（如 20 小时监控）。
3. **自动扫描与处理**: 系统自动定位有目标的区间，仅对关键时段调用大模型分析。
4. **语义搜索**: 输入自然语言描述，如"骑电动车戴白色头盔的人"。
5. **精准定位**: 系统返回 5-10 秒视频切片及分析报告。

## 分片策略

系统支持两种分片模式，可在前端实时切换：

- **智能事件扫描模式 (推荐)**: 
    - **原理**: 利用本地 YOLO 模型对视频流进行 1fps 粗扫，记录目标轨迹坐标。
    - **优势**: 自动聚合连续事件，仅对“有人的时段”进行按需截取，处理效率提升数倍。
- **传统固定窗口模式**: 
    - **参数**: 8-10 秒固定窗口，5 秒步长重叠。
    - **过滤**: 结合 OpenCV 背景减除算法，过滤静止背景。
